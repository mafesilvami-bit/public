# public
What tools you installed
***Cursor***
***Claude Code for VS Code***: via Cursor Extensions marketplace
- **Codex** — AI extension installed via Cursor Extensions marketplace
- **Git** — Version control system, required for repository management
— What steps you completed
1. Downloaded and installed Cursor IDE from cursor.com
2. Searched for "Claude Code" in Cursor Extensions and installed Claude Code for VS Code
3. Searched for "Codex" in Cursor Extensions and installed the extension
4. Created a public GitHub repository
5. Installed Git for Windows to enable terminal-based repository management
6. Cloned the repository locally using `git clone`
7. Opened the repository folder in Cursor
8. Created and edited this README.md file
9. Committed and pushed changes to GitHub

— What issues you ran into and how you solved them
Issue 1: `git clone` command not recognized in terminal**  
The terminal returned a `CommandNotFoundException` error when running `git clone`.  
**Solution:** Git was not installed on the machine. Downloaded and installed Git for Windows from git-scm.com, restarted Cursor, and the command worked correctly.

Issue 2: "Claude Code" official extension not available**  
The task referenced a "Claude Code" add-on for Cursor, but there is no official Anthropic extension by that exact name in the marketplace.  
**Solution:** Installed "Claude Code for VS Code," the closest available extension.
Issue 3: During this setup process, I used Claude as an AI assistant to troubleshoot errors and clarify the steps, particularly when Git was not recognized in the terminal and when identifying the correct extensions to install.
